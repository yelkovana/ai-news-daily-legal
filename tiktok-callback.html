    What is this?
    
This is the OAuth redirect page for the AI Tries Vines TikTok developer app. TikTok sends an authorization code here after you approve the connection. You copy it into the local Python script which exchanges it for an access token. The token is stored locally on your computer — this page never sees or stores it.
